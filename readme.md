## Flashtag

#### dev instructions

 - clone all the flashtag repos to a common parent dir
 
 - composer install
 
 - copy `.env.example` to `.env`
 
 - `php artisan key:generate`

 - `vendor/bin/homestead make`

 - set up your `Homestead.yaml` to look similar to the example.

 - `vagrant up`
